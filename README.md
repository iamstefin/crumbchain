# CrumbChain

A File Crumbing CLI Written in Go

# Installation

```console
$ crumbchain -help
Usage of crumbchain:
  -c string
    	Do the crumbing
  -i string
    	Check the integrity
  -j string
    	Join the crumbs
  -s int
    	Set the size of the crumbs (default 1)
```
